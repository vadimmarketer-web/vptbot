# KB-бот техподдержки SmartVPT

Один бот техподдержки на всех клиентов SmartVPT. Снимает 80% повторяющихся вопросов («как сделать X»), эскалирует только то, что не нашёл в KB.

## Зачем сейчас (а не после 5+ клиентов)
Изначально планировали запускать при 5+ клиентах. Стартанули раньше — Вадим + Лёва решили начать MVP сейчас, чтобы:
- параллельно собирать KB на двух действующих клиентах (Барнаул + Дублёр) с боевыми вопросами;
- к моменту 3-го клиента иметь рабочий бот, а не сырьё.

## Концепция
Подробно: `smartvpt-vault/07-техподдержка/концепция.md` (в продуктовом vault'e).

Кратко:
```
Клиент задаёт вопрос
    ↓
Бот ищет в KB (Obsidian → vector store)
    ↓ confidence ≥ 0.7  →  Бот отвечает
    ↓ confidence < 0.7   →  Эскалация в наш чат, клиенту бот НЕ отвечает (не врём)
```

## Что уже делаем (с этого момента)
**Каждый ответ клиенту** в чате → копируем в `smartvpt-vault/07-техподдержка/kb/03-faq/<тема>.md`.

К моменту запуска MVP у нас должно быть 50+ выстраданных заметок (а не 50 заметок «из головы»).

## Что НЕ делаем
- Не даём боту SSH-доступ к серверам клиентов (attack surface при 20+ клиентах).
- Не врём клиенту при низкой уверенности — лучше эскалировать без ответа.
- Не пилим под одного клиента — KB универсальная для всех.
