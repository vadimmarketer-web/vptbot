# ADR-003 — Google Sheets как источник лидов

## Контекст
Альтернативы: завести в боте свою CRM, либо интегрироваться с готовой CRM клуба (1С, AmoCRM, Bitrix), либо использовать Google Sheets.

## Решение
**Google Sheets.** Менеджер клуба заносит лиды в таблицу, бот импортирует.

## Почему
- **Менеджеры уже работают с таблицами** — нулевая кривая обучения.
- **Не привязываемся к CRM клуба** — у разных клиентов разные системы (или нет вовсе).
- **Простота интеграции** — Google Sheets API стабильный, бесплатный, документированный.
- **Прозрачность** — клиент видит свои данные, не зависит от нашего интерфейса.

## Последствия
- ✅ Универсально для всех клиентов.
- ✅ Если клиент уйдёт — данные у него остались в его таблице.
- ⚠️ Нет real-time push — нужно поллить таблицу с интервалом.
- ⚠️ Менеджер может что-то сломать в формате — нужны валидаторы при импорте.
- ⚠️ Лимиты Google Sheets API (квоты на чтение) — следить, при росте до 10+ клиентов проверить.
