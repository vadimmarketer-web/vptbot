# Runbook: подключение нового клуба

## Предусловия
- Подписан договор (возмездное оказание услуг). Шаблон у Ильи.
- Согласован тариф (см. `04-финансы/бизнес-модель.md`).
- У клиента есть юрлицо для регистрации бота в MAX.

## Шаги

### 1. Инфраструктура
- [ ] Заказать VPS NetAngels (минимальная конфигурация под бот; уточнить у Лёвы конкретные характеристики).
- [ ] Включить годовую оплату — даёт скидку 10% (проксируется клиенту).
- [ ] Завести поддомен (если нужен).
- [ ] Поднять `systemd`-юнит для бота, настроить автостарт.
- [ ] Настроить ротацию `journalctl`.

### 2. Боты (на стороне клиента)
- [ ] Клиент регистрирует **MAX-бота** на своё юрлицо. Получает токен, передаёт нам.
- [ ] Создаётся **Telegram-бот** в BotFather (можно на нашей стороне для удобства, но лучше на стороне клиента — см. [[adr-004-юрлицо-клиента-в-MAX]]).
- [ ] Токены кладутся на VPS клиента (никаких shared-секретов между клубами).

### 3. Google Sheets
- [ ] Клиент создаёт таблицу под формат бота.
- [ ] Сервисный аккаунт Google добавляется в таблицу с правами «Редактор» (только своей таблицы клиента).
- [ ] Проверить, что бот видит таблицу и читает.

### 4. Деплой кода
- [ ] `git clone git@github.com:vadimmarketer-web/vptbot.git`
- [ ] Конфиг под клиента (токены, ID таблицы, имя клуба) — отдельным файлом, не в репо.
- [ ] `systemctl start <unit>` + `journalctl -u <unit> -f` для проверки.

### 5. Онбординг команды клиента
- [ ] Создать первого администратора одной ссылкой через бот.
- [ ] Админ создаёт координатора, координатор — тренеров.
- [ ] Прислать видео-инструкции (по ролям). NB: **карту-схему архитектуры НЕ показываем**.
- [ ] SLA: критическая ошибка — 1 день, пожелание — 3 дня.

### 6. Финансы
- [ ] Выставить счёт (setup + первая подписка, или год вперёд).
- [ ] Зафиксировать клиента в `03-клиенты/<имя>.md`.

## Проверка перед сдачей
- Менеджер заносит тестового клиента в таблицу → бот видит.
- Координатор распределяет → тренер получает.
- Тренер ведёт по этапам → статусы обновляются.
- Владелец видит статистику.
- Просрочка SLA → автовозврат в общий пул.
