# SmartVPT — описание продукта

**Сайт:** http://smartvpt.ru/ (заявки → бот в TG → группа проекта)

## Что это

Telegram/MAX-бот для управления вводными персональными тренировками (ВПТ) в фитнес-клубах.

## Как работает

```
Менеджер заносит лида в Google Sheets
    ↓
Бот подхватывает и шлёт координатору
    ↓
Координатор распределяет тренеру (с учётом загрузки 🟢🟡🔴)
    ↓
Тренер ведёт по этапам: звонок → договорённость → ВПТ → продажа
    ↓
Руководитель видит реальную конверсию в реальном времени
```

CRM не нужна — работает поверх Google Sheets.

## Три роли

### Владелец (Администратор)
- Общая статистика по клубу
- Создание координаторов одной ссылкой
- Управление командой
- Тест-режимы

### Координатор
- Распределение клиентов из таблицы
- Контроль нагрузки тренеров (🟢 мало / 🟡 нормально / 🔴 много)
- Возвраты, переназначения
- Учёт отпусков

### Тренер
- Получает назначения в мессенджере
- Пошаговое ведение клиента (звонок → ВПТ → продажа)
- Может добавлять своих клиентов
- Видит личную статистику

## Ключевые механики

- **SLA-таймеры** с автовозвратом клиента в общий пул при просрочке
- **Отказы** с фиксацией причины
- **Отчёты по неактивным тренерам** (3+ дня без действий)
- **Уведомления** о нарушенных дедлайнах
- **Лимит дедлайна:** конец рабочего дня (20:00) — если позже распределили, уже не ставим в этот же день

## Стек

- Backend: Python (Лёва)
- Хранение: разные серверы под клиентов на NetAngels (по серверу на клуб)
- Фронт: Telegram Bot API, MAX Bot API
- Источник лидов: Google Sheets (импорт через бот)

## Кейс эксперта (Илья Савельев, Крым)

В сети клубов в Крыму:
- **+15–20%** доходимости до ВПТ
- **+15–20%** результативности ВПТ
- **↑** рост продаж персонального тренинга

## Что НЕ делаем

- Не превращаем в мобильное приложение (бы делали другой ценник 50–100к/мес, не 5–7)
- Не даём клиентам выпадающие списки ВПТ/ПТ — пишут в комментариях
- Не показываем клиентам внутреннюю архитектуру (карта-схема) — они начинают «дорабатывать под себя»
- Не делаем кастом под каждого клиента бесплатно — кастом продаётся отдельно
