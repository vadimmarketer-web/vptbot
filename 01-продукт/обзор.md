# SmartVPT — что делает бот

Telegram/MAX-бот для управления вводными персональными тренировками (ВПТ) в фитнес-клубах. CRM не нужна — работает поверх Google Sheets.

## Поток
```
Менеджер заносит лида в Google Sheets
    ↓
Бот подхватывает и шлёт координатору
    ↓
Координатор распределяет тренеру (с учётом загрузки 🟢🟡🔴)
    ↓
Тренер ведёт по этапам: звонок → договорённость → ВПТ → продажа
    ↓
Руководитель видит реальную конверсию в реальном времени
```

## Три роли
- **Владелец (Администратор):** общая статистика по клубу, создание координаторов одной ссылкой, управление командой, тест-режимы.
- **Координатор:** распределение клиентов, контроль нагрузки тренеров (🟢/🟡/🔴), возвраты, переназначения, учёт отпусков.
- **Тренер:** получает назначения, пошаговое ведение клиента (звонок → ВПТ → продажа), может добавлять своих клиентов, личная статистика.

## Ключевые механики
- SLA-таймеры с автовозвратом клиента в общий пул при просрочке.
- Отказы с фиксацией причины.
- Отчёты по неактивным тренерам (3+ дня без действий).
- Уведомления о нарушенных дедлайнах.
- Лимит дедлайна: конец рабочего дня (20:00) — если позже распределили, не ставим в этот же день.

## Принципы (что НЕ делаем)
- Не превращаем в мобильное приложение (другой ценник, другой продукт).
- Не даём клиентам выпадающие списки ВПТ/ПТ — пишут в комментариях.
- Не показываем клиентам внутреннюю архитектуру (карта-схема) — копируют.
- Кастом под клиента — только за отдельные деньги, не бесплатно.
