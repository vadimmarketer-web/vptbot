# vptbot — vault разработки

Obsidian-хранилище **кодовой стороны** бота SmartVPT: архитектура, дев-бэклог, релизы, инциденты, runbooks.

Продуктовая, клиентская, юридическая и финансовая часть — в **отдельном vault** `smartvpt-vault` (репо `vadimmarketer-web/smartvpt`).

| Что | Где |
|---|---|
| Код бота | `~/.claude-lab/shared/vptbot/` → `git@github.com:vadimmarketer-web/vptbot.git` |
| Продукт / клиенты / финансы / юр / маркетинг | `smartvpt-vault` (репо `smartvpt`) |
| Этот vault | дев-сторона: что чинить, как деплоить, что помнить |

## Структура
- `00-мета/` — правила vault, лог
- `01-продукт/` — короткая выжимка продукта + ссылки в `smartvpt-vault`
- `02-команда/` — кто пишет код, кто тестит, кто ревьюит
- `03-задачи/` — дев-бэклог (баги + фичи), берётся из встреч + чатов
- `04-архитектура/` — стек, схемы, ADR, runbooks
- `05-релизы/` — changelog, планы релизов
- `06-инциденты/` — постмортемы, известные риски
- `07-kb-бот/` — отдельный сабпродукт: KB-бот техподдержки клиентов (план MVP, архитектура, процесс сбора FAQ)
- `99-исходники/` — выгрузки, скрины
